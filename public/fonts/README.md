# Polices LA MESA — titre « LA MESA »

## Lumios Marker (recommandé)

Police **Lumios Marker** par My Creative Land (Elena Genova) — style marker / sign pen, proche de ton visuel Canva.

[Lumios Marker sur My Creative Land](https://mycreativeland.com/family/lumios-marker/)

### Installation

1. Obtiens la police (achat web ou licence Canva Pro si tu l’y utilises déjà)
2. Place **l’un** de ces fichiers ici :
   - `public/fonts/lumios-marker.woff2` (recommandé)
   - ou `public/fonts/lumios-marker.otf`
3. Recharge le site

Sans fichier, le site utilise **Caveat** (Google Fonts) en attendant — plus doux que Permanent Marker.

### Convertir OTF → WOFF2

Si tu n’as qu’un `.otf`, tu peux le convertir avec [CloudConvert](https://cloudconvert.com/otf-to-woff2) ou [Font Squirrel Webfont Generator](https://www.fontsquirrel.com/tools/webfont-generator).
